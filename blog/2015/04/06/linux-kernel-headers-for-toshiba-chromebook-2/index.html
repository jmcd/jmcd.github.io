Linux Kernel (3.10.18) Headers for Toshiba Chromebook 2

Apr 6th, 2015 5:15 pm

If you want to install Virtualbox on a crouton chroot, you need kernel headers so that Virtualbox can compile modules.

My Toshiba Chromebook 2 (CB30-B-104) came with a 3.10.18 kernel. Building the headers can be a bit of a pain, so I share them here.
